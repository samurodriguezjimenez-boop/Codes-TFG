# Codes-TFG
Quantum states based in neural networks for many body problems in condensed-matter physics
Estdos cuánticos basados en redes neuronales para resolver problemas de muchos cuerpos en física de la materia condensada

Librerías imprescindibles: NumPy, SciPy, PyTorch, Matplotlib
Librerías recomendadas que aparecen: Scikitlearn, NetKet
